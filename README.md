## Code for Task1 is in [src/first-task](https://github.com/danielprzybytniewski/frontend-tasks/tree/main/src/first-task) folder.

## Installation

```bash
$ git clone https://github.com/danielprzybytniewski/frontend-tasks.git
$ npm install
```

## Running the app

```bash
# Development mode
$ npm run dev

# Production build
$ npm run build
$ npm run preview
```

```bash
# Run all tests
$ npm run test

# Watch mode for tests
$ npm run test:watch

# Test coverage report
$ npm run test:coverage
```
